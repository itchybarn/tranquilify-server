# Tranquilify Server

The backend API for Tranquilify, a misophonia headphones project. Built with FastAPI, PostgreSQL, and Docker.

## Prerequisites

Ensure everything on this list is as-stated before beginning the setup.

- **Docker** running

## First-time setup

1. Clone the repo.
2. Copy the env template:
   ```bash
   cp .env.example .env
   ```
